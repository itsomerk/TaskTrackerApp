# Task Tracker Application

This project is a task management system developed using C# Windows Forms and MSSQL.

## Features
- Add Task
- Delete Task
- Mark as Completed
- Mark as Not Completed

## Technologies
- C#
- Windows Forms
- MSSQL

## How to Run
1. Run the TaskTrackerDB.sql file in SQL Server
2. Open the project in Visual Studio
3. Run the application
