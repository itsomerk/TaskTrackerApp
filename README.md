# Task Tracker Application

This project is a task management system developed using C# Windows Forms and SQLite. It allows users to manage their daily tasks efficiently with a simple and user-friendly interface.

## Features
- Add Task
- Delete Task
- Mark as Completed
- Mark as Not Completed
- View Tasks in a table format

## Technologies
- C#
- Windows Forms
- SQLite

## How to Run
1. Open the project in Visual Studio
2. Restore NuGet packages (if needed)
3. Run the application

Note: The SQLite database (tasks.db) is created automatically when the application is executed.
